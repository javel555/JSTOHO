■インストールとアンインストール==============================
インストールはCDからデータをお使いのPCにコピーするだけです。
アンインストールは全部削除すればOKです。
レジストリには手を入れておりません。

■ストーリー====================================================
とある国に人々の感情を司る魔法の力をもった希望の面と、
面の力を操ることのできるこころ姫がいました。

ところがある日、王宮に招待されていた隣国のこいし姫が
希望の面を盗み出してしまいました。

希望の面が失われてはいずれ国中の人々の感情が暴走してしまいます。
こころ姫は希望の面を取り戻すため、仲間とともに冒険の旅にでるのでした。


■操作方法(キーボード操作)=========================================
・十字キー(やじるしキーorゲームパッド)
・決定キー(Zキーorゲームパッド1)
・キャンセキー(Xキーorゲームパッド2)
・ゲーム終了(ESCキー)


■ゲームの開始方法============================================
フォルダの中にある プリンセスこころちゃん.exe を実行してください。
